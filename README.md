# Dify Easy Deploy Showcase

Static one-page showcase for the Dify Docker Compose deployment simplification PRs. The page includes a scrollable side-by-side comparison of the original full `.env.example` and the new minimal `.env.default`.

Open `index.html` directly, or publish this repository with any static hosting service.
